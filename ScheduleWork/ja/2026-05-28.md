# プライバシーポリシー — ScheduleWork

**施行日:** 2026年5月28日

開発者であるキム・ジテ（Jitae Kim、以下「開発者」）は、モバイルアプリケーション **「ScheduleWork」**（以下「本アプリ」）の利用者の個人情報を尊重し、日本の「個人情報の保護に関する法律（個人情報保護法／APPI）」その他の関連法令を遵守します。ScheduleWork はシフト・勤務スケジュールやタスクを管理するためのツールアプリです。

## 1. 取得する個人情報の項目および取得方法

本アプリはアカウント登録を必要としません。利用者が直接入力する情報、サービス利用中に自動的に生成される情報、および利用者がフィードバックを送信する際に送信される情報があります。

### 1-1. 利用者が入力する情報（端末内のみに保存）

| 区分 | 項目 |
|---|---|
| 勤務スケジュール情報 | 勤務スケジュール、勤務種別 |
| タスク・メモ情報 | タスク項目、日別メモ、気分の記録 |
| ワークスペース設定 | 利用者が構成したワークスペース・表示設定 |

これらの情報は **端末内の SwiftData のみに保存** され、**外部サーバーへ送信されません**。iCloud（CloudKit）同期は使用しないため、入力したスケジュール・メモのデータが端末を離れることはありません。ウィジェット表示のため同一端末内の App Group 領域に一部データが共有されますが、これも端末を離れません。

### 1-2. 自動的に取得する情報

| 区分 | 項目 | 取得主体 |
|---|---|---|
| 端末識別子 | IDFV、IDFA（広告トラッキングに同意した場合のみ） | Firebase Analytics、Google AdMob |
| 端末・環境 | OS の種類・バージョン、端末モデル、アプリバージョン、言語・地域設定 | Firebase Analytics、Firebase Crashlytics |
| 位置 | IP アドレスに基づく大まかな位置（国／地域レベル） | Firebase Analytics |
| 利用記録 | 画面の閲覧、機能の利用イベント | Firebase Analytics |
| 広告データ | 広告の表示・クリック、SKAdNetwork アトリビューション | Google AdMob、Apple SKAdNetwork |
| 診断・クラッシュデータ | 異常終了ログ、エラースタック、端末状態およびパフォーマンス指標 | Firebase Crashlytics |

### 1-3. フィードバック送信時に送信される情報

アプリ内のフィードバック機能で意見を送信した場合、以下の情報が Google Firestore（Google Cloud）サーバーへ送信・保存されます。アカウント・ログインがないため、利用者自身が入力しない限り氏名・メールアドレス等の識別情報は含まれません。

| 区分 | 項目 |
|---|---|
| フィードバック内容 | 利用者が記述した自由記述テキスト |
| 端末・アプリ環境 | 端末モデル、iOS バージョン、アプリバージョン、送信時刻（サーバータイムスタンプ） |

## 2. 利用目的

- 本アプリの中核機能の提供：勤務スケジュール・勤務種別・タスク・メモの作成・編集・表示
- ウィジェットによるスケジュール表示（端末内データのみ使用）
- サービス品質の改善および利用統計の分析（Firebase Analytics）
- アプリの安定性診断および不具合修正（Firebase Crashlytics）
- 広告の配信および効果測定（Google AdMob、Apple SKAdNetwork）
- 広告同意の管理（Google User Messaging Platform）
- 利用者フィードバックの受領およびサービス改善（Google Firestore）
- アプリ内課金の処理（Apple StoreKit）

## 3. 保有期間

| 項目 | 保有期間 |
|---|---|
| 利用者が入力したスケジュール・メモデータ | 端末内、利用者が削除またはアプリを削除するまで |
| フィードバック内容および環境情報 | サービス改善の目的に必要な間保持し、利用者の請求により削除（下記第5項を参照） |
| 分析データ（イベントログ） | 取得日から最長14か月 |
| 広告識別子および広告履歴 | Google AdMob／Apple SKAdNetwork のポリシーに従う |
| クラッシュ・診断データ | Firebase Crashlytics のポリシーに従う（通常90日） |

## 4. 第三者提供および処理の委託

本アプリは、自動取得情報およびフィードバック情報の一部の処理を以下の委託先に委託します。**利用者が入力したスケジュール・メモデータは、いかなる第三者にも提供されません。**

| 委託先 | 委託業務 | 委託項目 |
|---|---|---|
| Google LLC (Firebase Analytics) | アプリ利用分析 | 端末識別子、イベントログ、IP |
| Google LLC (Firebase Crashlytics) | クラッシュ・診断分析 | 異常終了ログ、端末・アプリ状態 |
| Google LLC (Firebase Firestore) | 利用者フィードバックの保存 | フィードバック内容、端末モデル、iOS・アプリバージョン、タイムスタンプ |
| Google LLC (AdMob) | 広告配信および測定 | 広告識別子、端末情報、IP、広告履歴 |
| Google LLC (User Messaging Platform) | 広告同意の管理（EU/EEA 等） | 同意状態、端末識別子 |
| Apple Inc. | アプリ配布、アプリ内課金処理、広告アトリビューション（SKAdNetwork） | アトリビューション情報、購入取引情報 |

委託先は日本国外（主に米国）のサーバーでデータを処理する場合があります。外国にある第三者への提供にあたっては、個人情報保護法の定めに従います。

## 5. 利用者の権利および行使方法

利用者は以下の権利をいつでも行使できます。

- **自身が入力したスケジュール・メモデータの開示・訂正・削除**：アプリ内で直接編集・削除可能です（開発者は当該データの複製を保有しないため、開発者への別途請求は不要です）
- **アプリ全データの削除**：アプリを削除すると端末内のすべてのスケジュール・メモデータが除去されます
- **フィードバック情報の開示・削除**：送信したフィードバックには識別子がなく個別特定は困難ですが、送信時刻・内容をお知らせいただければ確認のうえ削除します
- **トラッキング同意の撤回**：iOS の[設定]→[プライバシーとセキュリティ]→[トラッキング]→[ScheduleWork]をオフ
- **広告識別子のリセット**：iOS の[設定]→[プライバシーとセキュリティ]→[Apple の広告]
- 委託先が保有する自動取得情報の**開示・削除・利用停止**の請求は、下記連絡先までお送りください。正当な請求には遅滞なく対応します。

## 6. 安全管理措置

- 利用者が入力したスケジュール・メモデータは外部サーバーへ送信されず、端末内でのみ処理されます
- iCloud（CloudKit）同期を使用しないため、入力データはクラウドに同期されません
- フィードバック送信および第三者 SDK との通信はすべて HTTPS（TLS）暗号化チャネルを使用します
- フィードバックにはアカウント・識別子が含まれず、送信者を特定しません
- 取得する識別子は匿名・仮名化された形式でのみ利用されます

## 7. お問い合わせ・フィードバック機能

本アプリには、利用者が開発者へ意見を送信できるフィードバック機能が含まれています。フィードバックを送信すると、記述内容と端末モデル・iOS バージョン・アプリバージョン・送信時刻が Google Firestore に保存され、開発者がこれを閲覧してサービス改善に利用します。フィードバック本文に含める個人情報は利用者ご自身が決定するため、氏名・連絡先など本人を特定できる情報は必要な場合のみご入力ください。
