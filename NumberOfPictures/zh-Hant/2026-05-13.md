# 隱私政策 — Number on Photos (照片編號 / 사진에 숫자 입력)

**生效日期:** 2026年5月13日

開發者 Jitae Kim(以下簡稱「我們」)尊重行動應用程式 **「Number on Photos」**(以下簡稱「本應用程式」)使用者的個人資料。本政策說明本應用程式收集、使用及分享哪些資料,以及您擁有的權利。本政策以歐盟《一般資料保護規則》(GDPR)為全球基準撰寫。

## 1. 資料控管者

**Jitae Kim**
電子郵件:**wlxo0401@gmail.com**

## 2. 收集的個人資料類別

本應用程式無需註冊帳號,我們不會直接要求您提供個人資料。在您使用本應用程式的過程中,以下資料可能會被自動收集:

| 類別 | 範例 |
|---|---|
| 裝置識別碼 | IDFV;IDFA(僅在您透過 App Tracking Transparency 同意時) |
| 裝置與環境 | 作業系統版本、裝置型號、應用程式版本、語言/地區設定 |
| 大致位置 | 基於 IP 位址推斷的國家/地區層級位置 |
| 使用資料 | 應用程式內事件、畫面瀏覽、功能互動 |
| 診斷資料 | 當機日誌、效能指標 |
| 廣告資料 | 廣告曝光、點擊、SKAdNetwork 歸因資訊 |
| 照片 | 僅在裝置本機讀取以執行編號功能 — **不會上傳至我們控制的任何伺服器** |

## 3. 處理目的與法律依據(GDPR 第6條)

| 目的 | 法律依據 |
|---|---|
| 提供照片編號核心功能 | 第6(1)(b)條 — 契約履行 |
| 分析與服務改進(Firebase Analytics) | 第6(1)(f)條 — 合法利益 |
| 當機診斷(Firebase Crashlytics) | 第6(1)(f)條 — 合法利益 |
| 個人化廣告(AdMob、IDFA) | 第6(1)(a)條 — 同意(透過 ATT) |
| 非個人化廣告 | 第6(1)(f)條 — 合法利益 |

## 4. 接收方 / 處理方

| 處理方 | 用途 | 所在國 |
|---|---|---|
| Google LLC(Firebase) | 分析、當機診斷 | 美國 |
| Google LLC(AdMob) | 廣告 | 美國 |
| Apple Inc.(SKAdNetwork) | 廣告歸因 | 美國 |

各方隱私政策:
- Google: https://policies.google.com/privacy

## 5. 國際資料傳輸

您的個人資料可能透過我們的處理方傳輸至美國。此類傳輸在適用時依據 **歐盟-美國資料隱私框架** 及/或 **標準契約條款(SCCs)** 進行。

## 6. 保留期間

| 項目 | 保留期間 |
|---|---|
| 分析資料 | 自收集日起最長 14 個月 |
| 當機日誌 | 自收集日起最長 90 天 |
| 廣告識別碼及廣告資料 | 依 Google AdMob 與 Apple SKAdNetwork 之政策 |
| 照片 | 不保留 — 僅在裝置本機處理 |

## 7. 您的權利

您有權:
- **存取** 您的個人資料
- **更正** 不正確的個人資料
- **刪除** 個人資料
- **限制處理**
- **資料可攜權**
- **反對** 基於合法利益的處理
- **隨時撤回同意**(不影響此前合法處理)
- 向您所在地的監管機關 **申訴**

行使上述權利,請來信 **wlxo0401@gmail.com**。

## 8. 撤回追蹤同意

- iOS 設定 → 隱私權與安全性 → 追蹤 → Number on Photos(關閉)
- iOS 設定 → 隱私權與安全性 → Apple 廣告 → 重置廣告識別碼
- iOS 設定 → 隱私權與安全性 → 照片 → Number on Photos(變更權限)
- 解除安裝本應用程式後,裝置上的所有本機資料將被刪除

## 9. 自動化決策

本應用程式 **不進行** 產生 GDPR 第22條所述法律或重大影響的自動化決策或剖析。

## 10. 兒童

本應用程式不以 16 歲以下兒童為對象,我們不會故意收集 16 歲以下兒童的個人資料。如您認為某兒童向我們提供了個人資料,請聯絡我們刪除。

## 11. 政策變更

我們可能不時更新本隱私政策。重大變更將在應用程式內或公告 URL 上通知。所有歷史版本依生效日期保留於同一儲存庫中。
