# 隱私政策 — Number on Photos (照片編號 / 사진에 숫자 입력)

**生效日期:** 2026年5月30日

開發者 Jitae Kim(以下簡稱「我們」)尊重行動應用程式 **「Number on Photos」**(以下簡稱「本應用程式」)使用者的個人資料。本政策說明本應用程式收集、使用及分享哪些資料,以及您擁有的權利。本政策以歐盟《一般資料保護規則》(GDPR)為全球基準撰寫。

**Jitae Kim**

## 2. 收集的個人資料類別

本應用程式無需註冊帳號,我們不會直接要求您提供個人資料。其中包括您在裝置本機建立/儲存的資料,以及由第三方 SDK 自動收集的資料。

### 2-1. 您在裝置上建立/儲存的資料(僅儲存於本機)

- **照片** — 您選擇的照片僅 **在裝置本機處理** 以執行編號功能,**絕不會上傳至我們控制的任何伺服器**。是否將合成結果儲存至相機膠卷(照片 App)由您選擇。
- **作業歷史** — 編號作業的記錄:建立時間、圖片名稱(參照)、輸入模式。**僅儲存於本機的 SwiftData 中**,不會傳輸至外部。

### 2-2. 自動收集的資料(由第三方 SDK / 系統收集)

以下資料 **並非由我們直接收集或儲存,而是由整合在本應用程式中的第三方 SDK 依各自目的自動收集與處理**。我們不會以可直接識別個人的形式儲存這些資料,僅查看彙總/匿名化後的結果。

| 類別 | 範例 | 收集主體 |
|---|---|---|
| 裝置識別碼 | IDFV | Firebase Analytics、Firebase Crashlytics、Google AdMob |
| 廣告識別碼 | IDFA(僅在您透過 App Tracking Transparency 同意時) | Google AdMob |
| 裝置與環境 | 作業系統版本、裝置型號、應用程式版本、語言/地區設定 | Firebase Analytics、Firebase Crashlytics、Google AdMob |
| 大致位置 | 基於 IP 位址推斷的國家/地區層級位置 | Firebase Analytics、Google AdMob |
| 使用資料 | 應用程式內事件、畫面瀏覽 | Firebase Analytics |
| 診斷/當機資料 | 當機日誌、效能指標 | Firebase Crashlytics |
| 廣告資料 | 廣告曝光、點擊 | Google AdMob |
| 廣告歸因 | 應用程式安裝/轉換歸因 | Apple SKAdNetwork |

## 3. 處理目的與法律依據(GDPR 第6條)

| 目的 | 法律依據 |
|---|---|
| 提供照片編號核心功能 | 第6(1)(b)條 — 契約履行 |
| 分析與服務改進(Firebase Analytics) | 第6(1)(f)條 — 合法利益 |
| 當機診斷(Firebase Crashlytics) | 第6(1)(f)條 — 合法利益 |
| 個人化廣告(AdMob、IDFA) | 第6(1)(a)條 — 同意(透過 ATT) |
| 非個人化廣告 | 第6(1)(f)條 — 合法利益 |
| 取得並管理廣告/追蹤同意(Google UMP) | 第6(1)(c)條 — 法律義務 / 第6(1)(f)條 — 合法利益 |

## 4. 接收方 / 處理方

**您在裝置上建立/儲存的照片和作業歷史不會與任何第三方共享。** 僅上述自動收集的資料由以下處理方處理:

| 處理方 | 用途 | 所在國 |
|---|---|---|
| Google LLC(Firebase Analytics, Crashlytics) | 分析、當機診斷 | 美國 |
| Google LLC(AdMob / UMP) | 廣告;廣告/追蹤同意管理 | 美國 |
| Apple Inc.(SKAdNetwork) | 廣告歸因 | 美國 |

各方隱私政策:
- Google: https://policies.google.com/privacy
- Apple: https://www.apple.com/legal/privacy/

## 5. 國際資料傳輸

您的個人資料可能透過我們的處理方傳輸至美國。此類傳輸在適用時依據 **歐盟-美國資料隱私框架** 及/或 **標準契約條款(SCCs)** 進行。

## 6. 保留期間

| 項目 | 保留期間 |
|---|---|
| 作業歷史 | 僅在本機 — 在您於應用程式內刪除或解除安裝本應用程式之前一直保留 |

## 8. 撤回追蹤/同意

- iOS 設定 → 隱私權與安全性 → 追蹤 → Number on Photos(關閉)
- 應用程式內 Google UMP 同意畫面 — 在顯示的同意畫面中可變更廣告/追蹤選擇
- iOS 設定 → 隱私權與安全性 → Apple 廣告 → 重置廣告識別碼
- iOS 設定 → 隱私權與安全性 → 照片 → Number on Photos(變更權限)
- 解除安裝本應用程式後,裝置上的所有本機資料(作業歷史、設定等)將被刪除

## 9. 自動化決策

本應用程式 **不進行** 產生 GDPR 第22條所述法律或重大影響的自動化決策或剖析。

## 10. 兒童

本應用程式不以 16 歲以下兒童為對象,我們不會故意收集 16 歲以下兒童的個人資料。如您認為某兒童向我們提供了個人資料,請聯絡我們刪除。

## 11. 政策變更

我們可能不時更新本隱私政策。重大變更將在應用程式內或公告 URL 上通知。所有歷史版本依生效日期保留於同一儲存庫中。
