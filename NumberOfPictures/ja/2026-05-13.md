# プライバシーポリシー — Number on Photos (写真に番号を入力 / 사진에 숫자 입력)

**施行日:** 2026年5月13日

開発者 Jitae Kim(以下「当方」)は、モバイルアプリケーション **「Number on Photos」**(以下「本アプリ」)におけるユーザーの個人情報を尊重し、日本の「個人情報の保護に関する法律」(APPI)その他関連法令を遵守します。本ポリシーは、本アプリがどのような情報を収集・利用・提供するか、ユーザーがどのような権利を持つかを説明します。

## 2. 取得する個人情報の項目

本アプリはアカウント登録を必要とせず、ユーザーから直接個人情報を入力させることはありません。ただし、本アプリの利用過程で以下の情報が自動的に収集されることがあります。

| 区分 | 項目 |
|---|---|
| 端末識別子 | IDFV、IDFA(App Tracking Transparency により同意した場合のみ) |
| 端末・環境情報 | OSバージョン、デバイスモデル、アプリバージョン、言語・地域設定 |
| 大まかな位置 | IPアドレスから推定される国・地域レベルの位置 |
| 利用履歴 | アプリ内イベント、画面閲覧、機能利用ログ |
| 診断情報 | クラッシュログ、パフォーマンス情報 |
| 広告データ | 広告表示・クリック、SKAdNetwork アトリビューション情報 |
| 写真 | 番号合成機能のために端末内でのみ読み込み — **当方のサーバーにアップロードされることはありません** |

## 3. 利用目的

- アプリの中核機能の提供(写真への番号合成、カメラロールへの保存)
- サービス品質改善および利用統計分析(Firebase Analytics)
- クラッシュ・不具合の診断(Firebase Crashlytics)
- 広告の配信および効果測定(Google AdMob、Apple SKAdNetwork)
- 法令上の義務の履行

## 7. ユーザーの権利

ユーザーは APPI 第28条以下に基づき、以下の権利を行使できます。

- **開示請求**(保有個人データの開示)
- **訂正・追加・削除**の請求
- **利用停止**または**消去**の請求
- **第三者提供の停止**の請求

権利行使は **wlxo0401@gmail.com** までご連絡ください。本人確認のうえ、遅滞なく対応します。

## 8. トラッキング同意の撤回方法

- iOS 設定 → プライバシーとセキュリティ → トラッキング → Number on Photos をオフ
- iOS 設定 → プライバシーとセキュリティ → Apple広告 → 広告識別子のリセット
- iOS 設定 → プライバシーとセキュリティ → 写真 → Number on Photos でアクセス権限を変更
- 本アプリをアンインストールすると、端末内のローカルデータはすべて削除されます

## 9. 安全管理措置

- 写真データは外部サーバーに送信されず、端末内でのみ処理されます
- 第三者 SDK との通信はすべて HTTPS(TLS)暗号化により保護されます
- 収集される識別子は匿名化・仮名化された形でのみ利用されます
