# 隐私政策 — Number on Photos (照片编号 / 사진에 숫자 입력)

**生效日期:** 2026年5月30日

开发者 Jitae Kim(以下简称"我们")尊重移动应用程序 **"Number on Photos"**(以下简称"本应用")用户的个人信息。本政策说明本应用收集、使用和共享哪些信息,以及您拥有的权利。本政策以欧盟《通用数据保护条例》(GDPR)为全球基准编写。

## 1. 数据控制者

**Jitae Kim**
邮箱:**wlxo0401@gmail.com**

## 2. 收集的个人数据类别

本应用无需注册账号,我们不会直接向您索取个人信息。其中包括您在设备本地创建/存储的数据,以及由第三方 SDK 自动收集的数据。

### 2-1. 您在设备上创建/存储的数据(仅存储于本机)

- **照片** — 您选择的照片仅 **在设备本地处理** 以执行编号功能,**绝不会上传到我们控制的任何服务器**。是否将合成结果保存到相机胶卷(照片 App)由您选择。
- **作业历史** — 编号作业的记录:创建时间、图片名称(引用)、输入模式。**仅存储在本机的 SwiftData 中**,不会传输至外部。

### 2-2. 自动收集的数据(由第三方 SDK / 系统收集)

以下数据 **并非由我们直接收集或存储,而是由集成在本应用中的第三方 SDK 按各自目的自动收集和处理**。我们不会以可直接识别个人的形式存储这些数据,仅查看汇总/匿名化后的结果。

| 类别 | 示例 | 收集主体 |
|---|---|---|
| 设备标识符 | IDFV | Firebase Analytics、Firebase Crashlytics、Google AdMob |
| 广告标识符 | IDFA(仅在您通过 App Tracking Transparency 同意时) | Google AdMob |
| 设备与环境 | 操作系统版本、设备型号、应用版本、语言/地区设置 | Firebase Analytics、Firebase Crashlytics、Google AdMob |
| 大致位置 | 基于 IP 地址推断的国家/地区级别位置 | Firebase Analytics、Google AdMob |
| 使用数据 | 应用内事件、屏幕浏览 | Firebase Analytics |
| 诊断/崩溃数据 | 崩溃日志、性能指标 | Firebase Crashlytics |
| 广告数据 | 广告展示、点击 | Google AdMob |
| 广告归因 | 应用安装/转化归因 | Apple SKAdNetwork |

## 3. 处理目的与法律依据(GDPR 第6条)

| 目的 | 法律依据 |
|---|---|
| 提供照片编号核心功能 | 第6(1)(b)条 — 合同履行 |
| 分析与服务改进(Firebase Analytics) | 第6(1)(f)条 — 合法利益 |
| 崩溃诊断(Firebase Crashlytics) | 第6(1)(f)条 — 合法利益 |
| 个性化广告(AdMob、IDFA) | 第6(1)(a)条 — 同意(通过 ATT) |
| 非个性化广告 | 第6(1)(f)条 — 合法利益 |
| 获取并管理广告/跟踪同意(Google UMP) | 第6(1)(c)条 — 法律义务 / 第6(1)(f)条 — 合法利益 |

## 4. 接收方 / 处理方

**您在设备上创建/存储的照片和作业历史不会与任何第三方共享。** 仅上述自动收集的数据由以下处理方处理:

| 处理方 | 用途 | 所在国 |
|---|---|---|
| Google LLC(Firebase Analytics, Crashlytics) | 分析、崩溃诊断 | 美国 |
| Google LLC(AdMob / UMP) | 广告;广告/跟踪同意管理 | 美国 |
| Apple Inc.(SKAdNetwork) | 广告归因 | 美国 |

各方隐私政策:
- Google: https://policies.google.com/privacy
- Apple: https://www.apple.com/legal/privacy/

## 5. 国际数据传输

您的个人数据可能通过我们的处理方传输至美国。此类传输在适用时依据 **欧盟-美国数据隐私框架** 和/或 **标准合同条款(SCCs)** 进行。

| 项目 | 保留期限 |
|---|---|

## 8. 撤回跟踪/同意

- iOS 设置 → 隐私与安全性 → 跟踪 → Number on Photos(关闭)
- 应用内 Google UMP 同意界面 — 在显示的同意界面中可更改广告/跟踪选择
- iOS 设置 → 隐私与安全性 → Apple 广告 → 重置广告标识符
- iOS 设置 → 隐私与安全性 → 照片 → Number on Photos(更改权限)
- 卸载本应用后,设备上的所有本地数据(作业历史、设置等)将被删除

## 9. 自动化决策

本应用 **不进行** 产生 GDPR 第22条所述法律或重大影响的自动化决策或画像分析。

## 10. 儿童

本应用不面向 16 岁以下儿童,我们不会有意收集 16 岁以下儿童的个人数据。如您认为某儿童向我们提供了个人数据,请联系我们删除。

## 11. 政策变更

我们可能不时更新本隐私政策。重大变更将在应用内或发布 URL 上公告。所有历史版本按生效日期保留在同一仓库中。
