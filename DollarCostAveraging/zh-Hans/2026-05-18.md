# 隐私政策 — 루프 (Loop)

**生效日期：** 2026年5月18日

本隐私政策说明移动应用程序 **"루프 - 적립식 투자 일지 작성 (Loop)"**（以下简称"本应用"）如何处理用户的个人数据。本应用由 Jitae Kim（金志泰，以下简称"开发者"）开发，是一款用于在设备本地记录、管理定投（分批买入）投资的工具应用。本政策以欧盟《通用数据保护条例》（GDPR）作为全球基准编写。

## 2. 个人数据的类别

本应用无需注册账户。

### 2-1. 您在应用中输入的数据（仅存储于设备本地）

- 标的信息：标的名称、券商名称、上市市场区分、目标数量
- 交易记录：买/卖区分、数量、交易日期
- 通知设置：按标的设置的提醒星期与时间

这些数据**仅存储在设备本地的 SwiftData 中**，**绝不会传输到我们运营的任何服务器**。本应用不提供任何云同步功能，因此您输入的投资数据不会离开您的设备。

### 2-2. 自动收集的数据

| 类别 | 示例 |
|---|---|
| 设备标识符 | IDFV；IDFA（仅在您通过 App 跟踪透明度同意时） |
| 设备与环境 | 操作系统版本、设备型号、应用版本、语言区域 |
| 大致位置 | 根据 IP 地址推断的国家/地区 |
| 使用数据 | 应用内事件、浏览的页面、功能交互 |
| 广告数据 | 广告展示、点击、SKAdNetwork 归因 |
| 诊断/崩溃数据 | 在发布版本中通过 Firebase Crashlytics 自动收集的崩溃日志、错误记录、性能指标 |

> 分析（Firebase Analytics）与崩溃诊断（Firebase Crashlytics）的数据**在开发（调试）版本中不会被收集**，仅在通过 App Store 分发的发布版本中收集。

| 目的 | 法律依据 |
|---|---|

| 您所设定的本地提醒通知 | 第 6(1)(b) 条 — 履行合同 |
| 分析与服务改进（Firebase Analytics） | 第 6(1)(f) 条 — 合法利益 |
| 崩溃/诊断报告（Firebase Crashlytics） | 第 6(1)(f) 条 — 合法利益 |

| 非个性化广告 | 第 6(1)(f) 条 — 合法利益 |
| 法律合规 | 第 6(1)(c) 条 — 法律义务 |

## 8. 加州居民（CCPA / CPRA）

如果您是加州居民，您有权：
- **知悉**收集哪些个人信息及其使用方式
- **删除**从您处收集的个人信息
- **更正**不准确的个人信息
- **拒绝**将个人信息用于跨情境行为广告的"共享"（在 CPRA 下，传递给 AdMob 的广告标识符可能构成"共享"）
- **限制**敏感个人信息的使用（我们不收集 CPRA 所定义的敏感个人信息）
- 行使权利时**不被歧视**

我们**不**为金钱对价**"出售"**个人信息。如需选择退出基于跟踪的广告，请在 ATT 提示中拒绝或在 iOS 设置中关闭跟踪。

如需提交 CCPA/CPRA 请求，请发送邮件至 **wlxo0401@gmail.com**。

## 9. 撤回跟踪/通知同意

- iOS 设置 → 隐私与安全 → 跟踪 → 루프（关闭开关）
- iOS 设置 → 隐私与安全 → Apple 广告 → 重置广告标识符
- iOS 设置 → 通知 → 루프 → 关闭通知

## 10. 通知（推送）

本应用根据您设定的按标的计划（星期与时间）发送**在您设备上生成的本地通知**。这些通知不通过我们运营的任何服务器或任何远程推送服务器；通知计划仅存储在您的设备上。您可以随时在 iOS 设置 → 通知 → 루프 中更改通知投递设置。

## 11. 应用内反馈邮件

本应用包含一项反馈功能，可打开 Apple Mail 应用以撰写消息。邮件通过 Apple Mail 应用**直接**发送，不经过我们运营的任何服务器。邮件的内容及其中包含的任何个人信息完全由您决定；一经发送，将由 Apple 的邮件基础设施和您的邮件服务商处理。

本应用还可能查询 Apple App Store 的公开应用信息以向您显示最新版本；此过程中不会收集或传输任何可识别个人身份的信息。

## 12. 自动化决策

本应用**不会**进行产生 GDPR 第 22 条所述法律或类似重大影响的自动化决策或画像。本应用不提供投资建议；所有条目均为用户自行记录的日志数据。

## 13. 儿童

本应用并非面向 16 岁以下儿童，我们不会在知情情况下收集 16 岁以下儿童（在适用 COPPA 的司法管辖区为 13 岁以下）的个人数据。如您认为儿童向我们提供了个人数据，请联系我们，我们将予以删除。

## 14. 本政策的变更

我们可能会不时更新本隐私政策。重大变更将在应用内或发布 URL 上公告。所有历史版本按生效日期保存于同一仓库。
