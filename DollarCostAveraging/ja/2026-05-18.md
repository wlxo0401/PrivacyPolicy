# プライバシーポリシー — 루프 (Loop)

**施行日:** 2026年5月18日

開発者であるキム・ジテ（Jitae Kim、以下「開発者」）は、モバイルアプリケーション **「루프 - 적립식 투자 일지 작성 (Loop)」**（以下「本アプリ」）の利用者の個人情報を尊重し、日本の「個人情報の保護に関する法律（個人情報保護法／APPI）」その他の関連法令を遵守します。本アプリは積立式（分割買付）投資の記録を端末内で保存・管理するためのツールアプリです。

## 1. 取得する個人情報の項目および取得方法

本アプリはアカウント登録を必要としません。利用者が直接入力する情報と、サービス利用中に自動的に生成される情報があります。

### 1-1. 利用者が入力する情報（端末内のみに保存）

| 区分 | 項目 |
|---|---|
| 銘柄情報 | 銘柄名、証券会社名、上場市場区分、目標数量など利用者が入力した値 |
| 取引履歴 | 売買区分、数量、取引日 |
| 通知設定 | 銘柄ごとの通知曜日・時刻など利用者が設定した通知スケジュール |

これらの情報は **端末内の SwiftData のみに保存** され、**開発者が運営するサーバー等の外部へ送信されません**。クラウド同期機能を提供しないため、入力した投資データが端末を離れることはありません。

### 1-2. 自動的に取得する情報

| 区分 | 項目 |
|---|---|
| 端末識別子 | IDFV、IDFA（広告トラッキングに同意した場合のみ） |
| 端末・環境 | OS の種類・バージョン、端末モデル、アプリバージョン、言語・地域設定 |
| 位置 | IP アドレスに基づく大まかな位置（国／地域レベル） |
| 利用記録 | 画面の閲覧、機能の利用イベント |
| 広告データ | 広告の表示・クリック、SKAdNetwork アトリビューション |
| 診断・クラッシュデータ | リリースビルドにおいて Firebase Crashlytics により自動的に取得される異常終了ログ、エラー記録、パフォーマンス指標 |

> 分析（Firebase Analytics）およびクラッシュ診断（Firebase Crashlytics）のデータは **開発（デバッグ）ビルドでは取得されず**、App Store で配信されるリリースビルドでのみ取得されます。

## 2. 利用目的

- 本アプリの中核機能の提供：銘柄・取引履歴の管理、積立式投資記録、目標数量の進捗表示
- 利用者が設定したスケジュールに基づくローカル通知の送出
- サービス品質の改善および利用統計の分析（Firebase Analytics）
- アプリの安定性診断および不具合修正（Firebase Crashlytics）
- 広告の配信および効果測定（Google AdMob、Apple SKAdNetwork）

## 3. 保有期間

| 項目 | 保有期間 |
|---|---|
| 利用者が入力した投資・取引・通知データ | 端末内、利用者が削除またはアプリを削除するまで |
| 分析データ（イベントログ） | 取得日から最長14か月 |
| クラッシュ・診断データ | Google Firebase Crashlytics のポリシーに従う（通常 最長90日） |
| 広告識別子および広告履歴 | Google AdMob／Apple SKAdNetwork のポリシーに従う |

## 4. 第三者提供および処理の委託

本アプリは、自動取得情報の一部の処理を以下の委託先に委託します。**利用者が入力した投資・取引データは、いかなる第三者にも提供されません。**

| 委託先 | 委託業務 | 委託項目 |
|---|---|---|
| Google LLC (Firebase Analytics) | アプリ利用分析 | 端末識別子、イベントログ、IP |
| Google LLC (Firebase Crashlytics) | クラッシュ・診断ログの収集 | 端末・環境情報、エラー・クラッシュログ |
| Google LLC (AdMob) | 広告配信および測定 | 広告識別子、端末情報、IP、広告履歴 |
| Apple Inc. | アプリ配布、広告アトリビューション（SKAdNetwork） | アトリビューション情報 |

委託先は日本国外（主に米国）のサーバーでデータを処理する場合があります。外国にある第三者への提供にあたっては、個人情報保護法の定めに従います。

委託先のプライバシーポリシー：
- Google: https://policies.google.com/privacy
- Google 広告: https://policies.google.com/technologies/ads
- Apple: https://www.apple.com/legal/privacy/

## 5. 利用者の権利および行使方法

利用者は以下の権利をいつでも行使できます。

- **自身が入力した投資・取引・通知データの開示・訂正・削除**：アプリ内で直接編集・削除可能
- **アプリ全データの削除**：アプリを削除すると端末内のすべての投資データが除去されます
- **トラッキング同意の撤回**：iOS の[設定]→[プライバシーとセキュリティ]→[トラッキング]→[루프]をオフ
- **広告識別子のリセット**：iOS の[設定]→[プライバシーとセキュリティ]→[Apple の広告]
- **通知権限の解除**：iOS の[設定]→[通知]→[루프] で通知をオフ
- 委託先が保有する自動取得情報の**開示・削除・利用停止**の請求は、下記連絡先までお送りください。正当な請求には遅滞なく対応します。

## 6. 安全管理措置

- 利用者が入力した投資・取引データは外部サーバーへ送信されず、端末内でのみ処理されます
- クラウド同期機能を提供しないため、投資データがクラウドへ送信されません
- 第三者 SDK との通信はすべて HTTPS（TLS）暗号化チャネルを使用します
- 取得する識別子は匿名・仮名化された形式でのみ利用されます

## 7. 通知（プッシュ）機能について

本アプリは、利用者が設定した銘柄ごとのスケジュール（曜日・時刻）に従って **端末内で生成されるローカル通知** を送出します。これらの通知は開発者のサーバーや外部のプッシュサーバーを経由せず、通知スケジュール情報は端末内のみに保存されます。通知の受信可否は iOS の[設定]→[通知]→[루프] でいつでも変更できます。

## 8. お問い合わせ・フィードバックメール送信機能

本アプリには、利用者が開発者へフィードバックを送信できるよう、メール作成機能（Apple Mail アプリの呼び出し）が含まれています。この機能は Apple Mail アプリを介して利用者ご自身が **直接** 発信する方式であり、開発者のサーバーを経由しません。メール本文に含まれる個人情報は利用者ご自身が決定し、送信後は Apple のメールインフラおよび利用者のメールアカウントのポリシーに従って処理されます。

また、本アプリは最新バージョン案内のために Apple App Store の公開アプリ情報を照会することがありますが、その過程で個人を特定できる情報は取得・送信されません。

## 9. 子どもの個人情報

本アプリは16歳未満の子どもを主たる対象とせず、16歳未満の子どもの個人情報を意図的に取得しません。保護者の方が子どもの個人情報が取得されたことに気づいた場合は、下記連絡先までご連絡ください。直ちに削除します。
