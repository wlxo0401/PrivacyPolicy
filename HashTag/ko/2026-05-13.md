# 복붙태그 — 개인정보 처리방침

**시행일자:** 2026년 5월 13일

김지태(이하 "개발자")는 모바일 애플리케이션 **"복붙태그 (CopyTag)"**(이하 "앱") 이용자의 개인정보를 중요시하며, 「개인정보 보호법」 등 관련 법령을 준수합니다.

## 1. 처리하는 개인정보 항목 및 수집 방법

앱은 별도의 회원가입이 필요 없으며, 해시태그 관리 기능을 위해 이용자가 직접 입력한 정보와 서비스 이용 과정에서 자동으로 생성되는 정보가 있습니다.

### 1-1. 이용자가 직접 입력하는 정보 (기기 내에만 저장)

| 구분 | 항목 |
|---|---|
| 태그 정보 | 해시태그 텍스트, 태그 그룹 이름, 태그 설명·메모 |
| 포스트 정보 | 이용자가 작성·복사한 포스트 텍스트, 첨부 태그 묶음 |
| 사용 이력 | 태그·포스트의 생성·복사 일시 등 앱 내 사용 기록 |

이 정보는 **기기 내 Realm 및 SwiftData에만 저장**되며, **외부 서버로 전송되지 않습니다**. iCloud(CloudKit) 동기화는 사용하지 않으므로, 이용자가 입력한 태그·포스트 데이터는 해당 기기를 벗어나지 않습니다.

### 1-2. 자동 수집 정보

| 구분 | 항목 |
|---|---|
| 기기 식별자 | IDFV, IDFA(광고 추적 동의 시에만) |
| 기기·환경 | OS 종류·버전, 기기 모델, 앱 버전, 언어·지역 설정 |
| 위치 | IP 주소 기반 대략적 위치(국가/지역 수준) |
| 이용 기록 | 화면 조회, 기능 사용 이벤트 |
| 광고 데이터 | 광고 노출·클릭, SKAdNetwork 어트리뷰션 |
| 진단·충돌 데이터 | Apple App Store Connect를 통해 시스템 레벨로 수집되는 비정상 종료 로그 및 성능 지표 (iOS [설정] → [개인정보 보호 및 보안] → [분석 및 향상] → "앱 개발자와 공유"를 활성화한 경우에 한함) |

## 2. 개인정보의 처리 목적

- 앱의 핵심 기능 제공: 태그·태그 그룹·포스트의 생성·편집·복사
- 사용 통계(차트) 표시 — 이용자 본인의 사용 기록을 기기 내에서 시각화
- 서비스 품질 개선 및 이용 통계 분석 (Firebase Analytics)
- 광고 제공 및 광고 효과 측정 (Google AdMob, Apple SKAdNetwork)
- 광고 동의 관리 (Google User Messaging Platform)
- 앱 안정성 진단 (Apple 시스템 크래시 보고)

## 3. 개인정보의 보유 및 이용 기간

| 항목 | 보유 기간 |
|---|---|
| 이용자가 입력한 태그·포스트 데이터 | 기기 내, 이용자가 삭제하거나 앱을 삭제할 때까지 |
| 분석 데이터(이벤트 로그) | 수집일로부터 최대 14개월 |
| 광고 식별자 및 광고 이력 | Google AdMob / Apple SKAdNetwork 정책에 따름 |
| 충돌·진단 데이터 | Apple App Store Connect 정책에 따름 |

## 4. 개인정보의 제3자 제공 및 처리 위탁

앱은 다음 수탁업체에 일부 자동 수집 정보의 처리를 위탁합니다. **이용자가 입력한 태그·포스트 데이터는 어떠한 제3자에게도 제공되지 않습니다.**

| 수탁업체 | 위탁 업무 | 위탁 항목 |
|---|---|---|
| Google LLC (Firebase Analytics) | 앱 사용 분석 | 기기 식별자, 이벤트 로그, IP |
| Google LLC (AdMob) | 광고 송출 및 측정 | 광고 식별자, 기기 정보, IP, 광고 이력 |
| Google LLC (User Messaging Platform) | 광고 동의 관리(EU/EEA 등) | 동의 상태, 기기 식별자 |
| Apple Inc. | 앱 배포, 광고 어트리뷰션(SKAdNetwork), 시스템 충돌 로그 수집 | 어트리뷰션 정보, 크래시·진단 로그 |

수탁업체는 한국 외 국가(주로 미국)에 위치한 서버에서 데이터를 처리할 수 있습니다.

## 5. 정보주체의 권리 및 행사 방법

이용자는 다음 권리를 언제든지 행사할 수 있습니다.

- **본인이 입력한 태그·포스트 데이터 열람·수정·삭제**: 앱 내에서 직접 수정·삭제 가능
- **앱 전체 데이터 삭제**: 앱 삭제 시 기기에 저장된 모든 태그·포스트 데이터가 제거됩니다
- **추적 동의 철회**: iOS [설정] → [개인정보 보호 및 보안] → [추적] → [복붙태그] 스위치 끄기
- **광고 식별자 재설정**: iOS [설정] → [개인정보 보호 및 보안] → [Apple 광고]
- **시스템 진단 공유 해제**: iOS [설정] → [개인정보 보호 및 보안] → [분석 및 향상] → "앱 개발자와 공유" 끄기
- 수탁업체가 보관 중인 자동 수집 정보에 대한 **열람·삭제·처리정지** 요청은 아래 연락처로 보내 주십시오. 정당한 요청은 지체 없이 처리합니다.

## 6. 개인정보의 안전성 확보 조치

- 이용자가 입력한 태그·포스트 데이터는 외부 서버에 전송되지 않으며, 기기 내에서만 처리됩니다
- iCloud(CloudKit) 동기화를 사용하지 않으므로, 입력 데이터가 클라우드에 동기화되지 않습니다
- 제3자 SDK와의 통신은 모두 HTTPS(TLS) 암호화 채널을 사용합니다
- 수집되는 식별자는 익명·가명 처리된 형태로만 활용됩니다

## 7. 사용자 문의·피드백 메일 발송 기능

앱에는 이용자가 개발자에게 피드백을 보낼 수 있도록 메일 작성 기능(Apple Mail 앱 호출)이 포함되어 있습니다. 이 기능은 Apple Mail 앱을 통해 이용자가 **직접** 발신하는 방식이며, 개발자의 서버를 거치지 않습니다. 메일 내용에 포함되는 개인정보는 이용자 본인이 결정하며, 발송 후에는 Apple의 메일 인프라 및 이용자의 메일 계정 정책에 따라 처리됩니다.
