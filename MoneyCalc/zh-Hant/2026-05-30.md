# 隱私政策 — 돈계산 (MoneyCalc)

**生效日期:** 2026年5月30日

開發者 Jitae Kim(以下簡稱「我們」)尊重行動應用程式 **「돈계산 - 모임 정산을 쉽고 빠르게」(MoneyCalc)**(以下簡稱「本應用程式」)使用者的個人資料。本政策以歐盟《一般資料保護規則》(GDPR)為全球基準撰寫。

**Jitae Kim**

## 2. 收集的個人資料類別

本應用程式無需註冊帳號。

### 2-1. 您輸入至本應用程式的資料(僅儲存於本機)

- 群組名稱、成員名稱及您輸入的任何標籤
- 結算記錄:項目名稱、金額、付款人資訊、日期、備註 等

這些資料 **僅儲存於本機的 SwiftData 中**,**絕不會傳輸至我們營運的任何伺服器**。iCloud(CloudKit)同步已刻意停用,您的結算資料不會離開您的裝置。小工具中顯示的資料透過 App Group 在同一裝置內共享,不會傳送至外部。

### 2-2. 自動收集的資料(由第三方 SDK / 系統收集)

以下資料 **並非由我們直接收集或儲存,而是由整合在本應用程式中的第三方 SDK 或 Apple 系統依各自目的自動收集與處理**。我們不會以可直接識別個人的形式儲存這些資料,僅查看彙總/匿名化後的結果。

| 類別 | 範例 | 收集主體 |
|---|---|---|
| 裝置識別碼 | IDFV | Firebase Analytics、Google AdMob |
| 廣告識別碼 | IDFA(僅在您透過 App Tracking Transparency 同意時) | Google AdMob |
| 裝置與環境 | 作業系統版本、裝置型號、應用程式版本、語言/地區設定 | Firebase Analytics、Google AdMob |
| 大致位置 | 基於 IP 位址推斷的國家/地區層級位置 | Firebase Analytics、Google AdMob |
| 使用資料 | 應用程式內事件、畫面瀏覽 | Firebase Analytics |
| 廣告資料 | 廣告曝光、點擊 | Google AdMob |
| 廣告歸因 | 應用程式安裝/轉換歸因 | Apple SKAdNetwork |
| 診斷/當機資料 | 當機日誌與效能指標(僅在您啟用 iOS 設定 → 隱私權與安全性 → 分析與改進 → 「與 App 開發者共享」時) | Apple(App Store Connect) |

> 本應用程式 **不** 使用獨立的當機診斷 SDK(例如 Firebase Crashlytics)。當機/診斷資料僅在 Apple 系統層級、且您選擇啟用時依上述方式收集。

### 2-3. 外部通訊 — 檢查應用程式最新版本

為向您告知最新版本,本應用程式會向 Apple 的公開 iTunes Lookup API(`itunes.apple.com`)**僅傳送應用程式識別碼(bundleId)** 以查詢 App Store 的最新版本資訊。此請求不包含任何可識別個人身分的資訊。

## 3. 處理目的與法律依據(GDPR 第6條)

| 目的 | 法律依據 |
|---|---|
| 提供核心功能(群組、成員、結算、小工具) | 第6(1)(b)條 — 契約履行 |
| 分析與服務改進(Firebase Analytics) | 第6(1)(f)條 — 合法利益 |
| 個人化廣告(AdMob、IDFA) | 第6(1)(a)條 — 同意(透過 ATT) |
| 非個人化廣告 | 第6(1)(f)條 — 合法利益 |
| 取得並管理廣告/追蹤同意(Google UMP) | 第6(1)(c)條 — 法律義務 / 第6(1)(f)條 — 合法利益 |
| 當機/診斷回報(Apple 系統層級,須選擇啟用) | 第6(1)(a)條 — 同意 |

## 4. 接收方 / 處理方

**您輸入至本應用程式的資料不會與任何第三方共享。** 僅上述自動收集的資料由以下處理方處理:

| 處理方 | 用途 | 所在國 |
|---|---|---|
| Google LLC(Firebase Analytics) | 分析 | 美國 |
| Google LLC(AdMob / UMP) | 廣告;廣告/追蹤同意管理 | 美國 |
| Apple Inc. | 應用程式發佈、SKAdNetwork 歸因、系統當機回報 | 美國 |

各方隱私政策:
- Google: https://policies.google.com/privacy
- Apple: https://www.apple.com/legal/privacy/

您的個人資料可能透過我們的處理方傳輸至美國。此類傳輸在適用時依據 **歐盟-美國資料隱私框架** 及/或 **標準契約條款(SCCs)** 進行。

## 8. 撤回追蹤/診斷同意

- iOS 設定 → 隱私權與安全性 → 追蹤 → 돈계산(關閉)
- 應用程式內 Google UMP 同意畫面 — 啟動時顯示的同意畫面中可變更廣告/追蹤選擇

本應用程式包含一項呼叫 Apple Mail 應用程式撰寫回饋郵件的功能。郵件 **由您直接** 透過 Apple Mail 應用程式發送,不經過我們營運的任何伺服器。郵件內容及其中包含的個人資料完全由您決定;發送後由 Apple 的郵件基礎設施及您的郵件供應商處理。
